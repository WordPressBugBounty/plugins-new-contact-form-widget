== Installation ==

1. Upload the plugin via **Plugins > Add New** or upload manually.
2. Activate the plugin.
3. Go to **Appearance > Widgets** and add the *Contact Form Widget* to any sidebar.
4. Or use the shortcode `[contact-form-widget]` in any post/page.
5. Customize your form settings and you're ready to go!

== Frequently Asked Questions ==

**Q. Does this plugin save form entries?**  
Yes, every submission is saved in the Query Table for easy access.

**Q. Can I send email notifications using SMTP?**  
Yes, Gmail SMTP and other SMTP providers are supported.

**Q. How do I add the form to a post or page?**  
Use the shortcode `[contact-form-widget]` or place the widget in any sidebar.

**Q. Can I customize the form fields?**  
Absolutely. You can rename or modify labels and styles from the settings panel.

**Q. Does it support premium features?**  
Yes. Premium features include more design options, extra templates, export tools, and more.
